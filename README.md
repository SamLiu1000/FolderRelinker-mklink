Ever find folders like '.cache' eating up tons of your C drive space? And when you try to move them, programs just start having all sorts of issues?

Well, mklink is a super reliable and common way in Windows to quickly move files or folders. 
You can stash the folder you want to move anywhere you like, and it'll leave a shortcut-like placeholder in the original location, keeping your programs happy.

And that's where Folder Relinker-mklink comes in. This tool was specifically created to simplify using mklink. It's got a user-friendly graphical interface, making it super quick and easy to manage all those mklink-created folders.

You can use Folder Relinker-mklink to easily relocate those pesky folders on your C drive, or manage your various web UI models, all to free up disk space. (won't work for Windows system files.)
